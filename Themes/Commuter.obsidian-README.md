# Commuter

> Vaseline for your eyes — without the smudge.

A dark, low-glare Obsidian theme crafted for reading on the train. Years of tuning for the strange lighting of the Göteborg–Stockholm commute: bright windows, dark tunnels, ceiling reflections, dim evenings, early mornings.

## Character

Neutral graphite base instead of pure black, soft gray text instead of hard white, and a restrained cyan accent for focus and orientation. The UI recedes so writing and structure come forward.

- Graphite backgrounds reduce glare and avoid muddy contrast.
- Muted foregrounds make long reading sessions easier on tired eyes.
- Cyan marks focus, links, active panes, and other navigational moments.
- Warm orange, red, green, and rose tones carry semantic meaning.

## Coverage

- editor and preview surfaces
- headings, links, tags, highlights, and blockquotes
- code blocks and inline code
- syntax colors
- sidebars, tabs, status bar, and navigation
- callouts, checkboxes, tables, modals, and graph view

Dark mode only — in light mode Obsidian's defaults apply.
